## 项目介绍以及实现结果

我们的项目是一个场景漫游——湖光山色，具体为一个第一视角的场景，角色可以用鼠标控制视角，键盘控制移动，在场景中四处漫游。场景包括山脉，湖泊，树木，并且有较为常见的实时天气效果。
